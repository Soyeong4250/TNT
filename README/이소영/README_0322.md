```
개발을 위한 IntelliJ Setting 중 JUnit 테스트가 실행이 되지 않는 에러가 발생했다.

👉 해결법
1️⃣ File > Settings... > Build,Execution,Deployment > Build Tools > Gradle로 이동
2️⃣ Build and run using과 Run tests using을 모두 IntelliJ IDEA로 변경
3️⃣ Apply > OK 클릭
```
